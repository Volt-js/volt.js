# Client-Side Integration

Learn how to integrate Volt.js with your frontend applications for a seamless full-stack experience with end-to-end type safety.

## What You'll Learn

In this section, you'll discover:

- Type-safe API client generation
- React integration with VoltProvider
- Data fetching with useQuery
- Mutations and form handling with useMutation
- Real-time updates with useRealtime
